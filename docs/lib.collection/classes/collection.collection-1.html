delete

  • delete(query: Query<Document>, options?: Partial<DeleteOptions<Document>>): Document[]

  • Removes documents that match the query. Emits DocumentNotification with operation DocumentOperation.DELETED which contains the deleted documents.

    example

    Delete document by id

     
    collection.delete('unique-id');

    Delete multiple documents

     
    // removes all documents having `fullName` equal to 'John'
const query = {
   fullName: 'John'
};
collection.delete(query);

    Parameters

    • query: Query<Document>
    • Optional options: Partial<DeleteOptions<Document>>

    Returns Document[]

    Deleted documents.

drop

  • drop(): void

  • Clear the Collection by removing all documents.
    Emits DocumentNotification with operation DocumentOperation.CLEARED and documents null.
    After dropping, collection becomes non-reactive, and no more DocumentNotification will be emitted. Therefore it's recommended to not reuse Collection object after this operation.

    Returns void

find

  • find(query?: Query<Document>, options?: Partial<FindOptions<Document>>): Document[]

  • Selects documents in a collection and returns them.
    Found documents are post-processed according to options.

    example

    Find all documents

     
    collection.find()

    Find document by id

     
    collection.find('unique-id')

    Find documents by indexed property

     
    // returns all documents having `birthYear` equals to 2000
collection.find(null, {
   index: {
       name: 'birthYear',
       value: 2000
   }
});

    Find documents and post-process them

     
    // returns all documents having `birthYear` greater than 2000, sorted by `firstName`
const query = { birthYear: { $gt: 2000 } };
const options = { sort: { firstName: SortDirection.ASCENDING } };
const matches = collection.find(query, options);

    Parameters

    • Optional query: Query<Document>
    • Optional options: Partial<FindOptions<Document>>

    Returns Document[]

replace

  • replace(query: Query<Document>, replacement: Document, options?: Partial<ReplaceOptions<Document>>): Document[]

  • Replace documents that match the query with the replacement.
    Emits 2 DocumentNotification when matches were replaced:

    1. DocumentOperation.DELETED with the matches that were removed.
    2. DocumentOperation.CREATED with the replacement.
    example

    Replace document by id

     
    collection.replace('unique-id', replacement);

    Replace multiple documents with a single one

     
     // replace all documents having `birthYear` greater than 2000
const query = { birthYear: { $gt: 2000 } };
collection.replace(query, replacement);

    Insert replacement if no matches found

     
     // no persons with name 'John' are present in the collection
const query = { firstName: 'John' };
collection.replace(query, replacement, { upsert: true });

    Parameters

    • query: Query<Document>
    • replacement: Document
    • Optional options: Partial<ReplaceOptions<Document>>

    Returns Document[]

update

  • update(query: Query<Document>, update: ObjMap, options?: Partial<UpdateOptions<Document>>): Document[]

  • Modifies an existing document or documents. The method can modify specific fields of an existing document or documents.
    When one of the indexed properties is updated, documents will be re-indexed with the new values of them.
    Emits DocumentNotification with operation DocumentOperation.UPDATED which contains the updated documents.

    example

    Update document by id

     
    const update = {
  $set: {
      birthYear: 2000
  }
};
// returns old document
collection.update('unique-id', update);

    Update multiple documents

     
    const update = {
  $inc: {
      salary: 100
  }
};
const query = {
  commits: {
      $gt: 25
  }
};
const options = {
  returnUpdates: true
};
// returns updated documents
collection.update(query, update, options);

    Reindex renamed property

     
    const update = {
   $rename: {
       oldIndexName: newName
   }
};
// will de-index renamed index property
collection.update('unique-id', update);

const bringBackIndex = {
  $unset: {
      newName: ''
  },
  $set: {
      oldIndexName: 'new-value'
  }
};
collection.update('unique-id', bringBackIndex);

    Parameters

    • query: Query<Document>
    • update: ObjMap
    • Optional options: Partial<UpdateOptions<Document>>

    Returns Document[]
