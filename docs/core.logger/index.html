@thermopylae/core.authentication

Logger for core Thermopylae modules.

Install

 
npm install @thermopylae/core.logger

Description

This package contains logging infrastructure used by Thermopylae core.* modules.

Logging is implemented with the help of winston npm package. The following transports are supported:

  • Console
  • File
  • Graylog2

Usage

Package exports a singleton instance of the Logger class, named LoggerManagerInstance.
Using its for method you can obtain logger for your module.
