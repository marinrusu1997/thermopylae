Enumeration AwaiterRole

CONSUMER

CONSUMER: = 1

Consumer which obtained lock, but operation has been started already by producer.
He needs to await on promise for it's fulfilling.

PRODUCER

PRODUCER: = 0

Producer which obtained lock and started operation.
He is responsible for fulfilling operation promise.
